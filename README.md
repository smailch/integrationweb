# ESPRIT CLUB — Dashboard (TailwindCSS)

## Prérequis
- Node.js 18+

## Installer
```bash
npm install
```

## Builder Tailwind
```bash
npm run build
```

## Watch (rebuild auto)
```bash
npm run watch
```

## Lancer un petit serveur local
```bash
npm run start
```
Puis ouvrir: http://127.0.0.1:5173

## Pages
- `/index.html` (Dashboard)
- `/liste-evenement.html` (Liste Evenement)
- `/chat-prive.html` (Chats privés)

Note: si le port `5173` est déjà utilisé, le serveur démarre automatiquement sur `5174`, `5175`, etc. (regarde l'URL affichée dans le terminal).
